Build Guide – Balcony Garden

Follow these steps to assemble your IoT-powered balcony garden from scratch. Allow 2–4 hours for the full setup.

1

Prepare the Planter Boxes

  1. Choose a balcony railing or floor spot that receives at least 6 hours of direct sunlight per day.
  2. Drill 4–6 drainage holes (6 mm) in the bottom of each planter box if not pre-drilled.
  3. Add a 2 cm layer of gravel or broken pottery at the base for extra drainage.
  4. Mix potting soil with 20% perlite, then fill each box to ~5 cm from the top.
  5. Mix in a measured dose of slow-release fertiliser granules as per package instructions.
2

Plant Your Seeds or Seedlings

  1. Make planting holes spaced 20–25 cm apart for most herbs; 30–40 cm for tomatoes.
  2. Place seedlings or seeds at the recommended depth (usually 2× seed diameter).
  3. Lightly firm soil around each plant and give an initial manual watering.
  4. Label each plant with a small marker stick.
3

Set Up the Irrigation System

  1. Place the 10 L water reservoir below or beside the planter boxes.
  2. Submerge the mini water pump in the reservoir and route the 4 mm drip tube up to the boxes.
  3. Split the tube with a T-connector to feed both planter boxes, then insert one or two drip emitters per box near each plant's root zone.
  4. Test the pump manually by applying 5 V directly – water should flow from all emitters.
  5. Insert one soil moisture sensor probe into the root zone of each box; route sensor wires to the electronics enclosure.
4

Wire the IoT Electronics

  1. Flash Raspberry Pi OS onto the MicroSD card using Raspberry Pi Imager. Enable SSH and Wi-Fi in the imager settings.
  2. Mount the Pi inside the waterproof enclosure; drill cable entry glands for sensor wires.
  3. Connect sensors to the Pi's GPIO header:
    • Soil Moisture: VCC → 3.3 V, GND → GND, AOUT → ADC (or use a MCP3008 SPI ADC)
    • DHT22: VCC → 3.3 V, GND → GND, DATA → GPIO4 (add 10 kΩ pull-up)
    • BH1750: VCC → 3.3 V, GND → GND, SDA → GPIO2, SCL → GPIO3
    • Relay IN1: GPIO17 (controls pump)
  4. Wire the pump's positive lead through the relay's NO (Normally Open) terminal.
  5. Double-check all connections with a multimeter before powering on.
5

Install Software & Configure Automation

  1. SSH into the Pi: ssh pi@<your-pi-ip>
  2. Update the system: sudo apt update && sudo apt upgrade -y
  3. Install Python libraries: 
    pip install gpiozero adafruit-circuitpython-dht smbus2
  4. Clone or copy the PlanAPlant control script to the Pi.
  5. Edit config.yaml to set your thresholds (moisture trigger, watering duration, etc.).
  6. Enable the service to run on boot: 
    sudo systemctl enable planaplant
sudo systemctl start planaplant
  7. (Optional) Install Mosquitto MQTT broker and connect Node-RED for a live dashboard.
6

Test & Calibrate the System

  1. Observe the Simulation Dashboard on this website to understand expected sensor ranges.
  2. Check live sensor readings via SSH: 
    python3 read_sensors.py
  3. Dry out the soil sensor probe in open air – it should read ~10%; submerge in water – it should read ~90%.
  4. Manually trigger the pump from the script; confirm all emitters discharge water evenly.
  5. Adjust the moisture threshold in config.yaml until the pump activates at the right time for your plants.
7

Ongoing Maintenance

  1. Weekly: Refill the water reservoir; check drip emitters for blockages.
  2. Monthly: Wipe sensor probes clean; verify relay and pump operation.
  3. Every 3 months: Top-dress with slow-release fertiliser; check wiring for corrosion.
  4. Seasonally: Swap out finished crops for new seedlings; recalibrate sensors if readings drift.
  5. Review the event log on the dashboard to spot unusual patterns (e.g., pump running too frequently can indicate a leak or damaged sensor).
